从 NumPy 1.17 开始，官方推荐使用新的随机数生成器（numpy.random.Generator），但传统的 np.random 函数依然可用且被大量代码使用。

1. 传统派
   * **种子设置**
     `np.random.seed(seed=None)`。设置随机数生成器的种子，确保结果可复现。
     `seed`:    int 或 None 随机种子，None 表示使用系统时间（默认）
   * **基础随机数生成**

     1. 生成指定维度随机数据。
        `np.random.rand(d0, d1, ..., dn)`生成 [0, 1) 均匀分布的随机数，维度由参数指定。
        ` d0, d1, ...`:    int 各维度大小，无参数时返回标量
        示例：
        ```python np.random.rand(3, 2)  # 3x2 数组，值在 [0,1) ```
     2. 生成指定维度标准正态分布（均值 0，方差 1）的随机数。
        ` np.random.randn(d0, d1, ..., dn)`
        示例：
        ```python np.random.randn(3)     # 3 个标准正态分布数 ```
     3. 生成 [0.0, 1.0) 均匀分布随机数，size 指定形状。
        `np.random.random(size=None)`
        参数：
        `size`： int 或 tuple，输出形状，None 返回标量

        ```python
            np.random.random((2, 3))   # 2x3 均匀分布数组 
        ```
     4. 生成随机整数（离散均匀分布）。
        `np.random.randint(low, high=None, size=None, dtype=int)`
        参数：

         | 参数    | 类型          | 说明                                                |
         | :------ | :------------ | :-------------------------------------------------- |
         | `low`   | int           | 下界（包含），若`high` 为 None，则范围为 `[0, low)` |
         | `high`  | int, optional | 上界（不包含）                                      |
         | `size`  | int 或 tuple  | 输出形状                                            |
         | `dtype` | dtype         | 输出类型，默认`int`                                 |

         示例：

           ```python
               np.random.randint(0, 10, size=5)      # 5 个 [0,10) 整数
               np.random.randint(5, size=(2, 3))     # 2x3 的 [0,5) 整数
           ```
     5. 生成均匀分布随机浮点数。
        `np.random.uniform(low=0.0, high=1.0, size=None)`

        ```python
                np.random.uniform(-1, 1, (3,))   # 3 个 [-1,1) 均匀数
        ```
   * 常见概率分布

     1. 正态分布（高斯分布）
        `np.random.normal(loc=0.0, scale=1.0, size=None)`
        参数：

        | 参数    | 类型      | 说明     |
        | :------ | :-------- | :------- |
        | `loc`   | float     | 均值     |
        | `scale` | float     | 标准差   |
        | `size`  | int/tuple | 输出形状 |
     
     2. 二项分布：
        n 次独立伯努利试验，成功概率 p，返回成功次数。
        `np.random.binomial(n, p, size=None)`
        参数：

        | 参数   | 类型                | 说明     |
        | :----- | :------------------ | :------- |
        | `n`    | int 或 array_like   | 试验次数 |
        | `p`    | float 或 array_like | 成功概率 |
        | `size` | int/tuple           | 输出形状 |
     3. 泊松分布
        事件发生率为 λ，返回单位时间内事件发生次数。
        ```python
            np.random.poisson(5, 10)   # 10个泊松样本，λ=5
        ```
     4. 指数分布  
        （尺度参数 β，均值 = β）。 
        `np.random.exponential(scale=1.0, size=None)`
           ```
           np.random.exponential(2, (3, 3))   # 尺度为2的指数分布
           ```
     5. 伽马分布     
        `np.random.gamma(shape, scale=1.0, size=None)`
        示例：
          ```python
                np.random.gamma(2, 2, 4)  # 形状参数2，尺度参数2，4个样本
          ```


2. 新 API：numpy.random.Generator
NumPy 1.17+ 引入了新的随机数生成器，基于 BitGenerator（如 PCG64），提供了更灵活、可分离的随机状态管理。使用 np.random.default_rng()
获取一个 Generator 对象。
示例:
```python
rng = np.random.default_rng(seed=42)   # 种子可选
```    
### 对应函数（通过 Generator 对象调用）
新 API 的方法命名更统一，通常为 `<distribution>`（如 `normal`、`uniform`），且增加了更多分布和功能。

| 传统 API                  | Generator 方法                                         | 说明                |
|:------------------------|:-----------------------------------------------------|:------------------|
| `np.random.seed`        | 通过 `default_rng(seed)` 设置                            | 推荐使用独立的 rng 对象    |
| `np.random.rand`        | `rng.random(size)`                                   | 返回 [0,1) 均匀分布     |
| `np.random.randn`       | `rng.standard_normal(size)` 或 `rng.normal(0,1,size)` | 标准正态              |
| `np.random.randint`     | `rng.integers(low, high, size)`                      | 随机整数（包含或不包含 high） |
| `np.random.uniform`     | `rng.uniform(low, high, size)`                       | 均匀分布              |
| `np.random.normal`      | `rng.normal(loc, scale, size)`                       | 正态分布              |
| `np.random.binomial`    | `rng.binomial(n, p, size)`                           | 二项分布              |
| `np.random.poisson`     | `rng.poisson(lam, size)`                             | 泊松分布              |
| `np.random.exponential` | `rng.exponential(scale, size)`                       | 指数分布              |
| `np.random.gamma`       | `rng.gamma(shape, scale, size)`                      | 伽马分布              |
| `np.random.choice`      | `rng.choice(a, size, replace, p)`                    | 随机选择              |
| `np.random.shuffle`     | `rng.shuffle(x)`                                     | 原地打乱              |
| `np.random.permutation` | `rng.permutation(x)`                                 | 返回打乱副本            |

**示例**：

```python
rng = np.random.default_rng(42)
print(rng.normal(0, 1, 3))           # 3个标准正态数
print(rng.integers(0, 10, (2, 3)))   # 2x3 整数
print(rng.choice(['a','b'], 5, p=[0.3,0.7]))
```

### 优势

- 可创建多个独立的随机数生成器，互不影响。
- 性能更好，支持更现代的 BitGenerator。
- 方法命名更清晰一致。
