| 函数名                       | 描述                     | 主要参数                                   | 返回值                    |
|:--------------------------|:-----------------------|:---------------------------------------|:-----------------------|
| **`pdf(x)`**              | 概率密度函数（PDF）            | `x`：数值或数组                              | 概率密度值                  |
| **`logpdf(x)`**           | 对数概率密度                 | `x`：数值或数组                              | 对数密度值                  |
| **`cdf(x)`**              | 累积分布函数（CDF）            | `x`：分位数                                | 概率值 P(X≤x)*P*(*X*≤*x*) |
| **`logcdf(x)`**           | 对数累积分布                 | `x`：数值或数组                              | 对数概率值                  |
| **`sf(x)`**               | 生存函数 = 1 - CDF         | `x`：分位数                                | 概率值 P(X>x)*P*(*X*>*x*) |
| **`logsf(x)`**            | 对数生存函数                 | `x`：数值或数组                              | 对数生存概率                 |
| **`ppf(q)`**              | 百分点函数（CDF 的反函数，即分位数函数） | `q`：概率值（0~1）                           | 对应分位数 x*x*             |
| **`isf(q)`**              | 逆生存函数（SF 的反函数）         | `q`：概率值（0~1）                           | 对应分位数                  |
| **`rvs(size)`**           | 生成随机变量                 | `size`：输出形状，`random_state`：随机种子        | 随机样本数组                 |
| **`stats(moments='mv')`** | 计算分布的统计量               | `moments`：指定要返回的矩（m 均值，v 方差，s 偏度，k 峰度） | 元组（均值, 方差, 偏度, 峰度）     |
| **`moment(n)`**           | n 阶非中心矩                | `n`：阶数                                 | 矩值                     |
| **`entropy()`**           | 分布的微分熵                 | 无                                      | 熵值                     |
| **`fit(data)`**           | 用最大似然估计拟合数据            | `data`：样本数据                            | 估计的 `(loc, scale)`     |
| **`interval(alpha)`**     | 包含分布 `alpha` 概率的区间     | `alpha`：置信水平（0~1）                      | 区间端点 `(left, right)`   |
