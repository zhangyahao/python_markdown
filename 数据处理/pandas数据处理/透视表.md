1. 函数

```python
python
import pandas as pd

pd.pivot_table(
    data,  # DataFrame 对象
    values=None,  # 需要聚合的数值列（可以多列）
    index=None,  # 行索引（分组键，可以多级）
    columns=None,  # 列索引（分组键，可以多级）
    aggfunc='mean',  # 聚合函数（默认均值），支持 'sum', 'mean', 'count', 'min', 'max' 等，也可传入自定义函数或字典
    fill_value=None,  # 填充空值的值
    margins=False,  # 是否添加行/列总计
    margins_name='All',  # 总计行列的名称
    dropna=True,  # 是否删除全为 NaN 的列
    observed=False  # 对于分类数据，是否只显示实际出现的组合
)
```

2. 参数详解
   | 参数 | 说明 |
   |:-----------------|:------------------------------------------------------------------------------------------------------------------------|
   | **`data`**       | 输入的 DataFrame。 |
   | **`values`**     | 要聚合的数值列名或列表。如果不指定，则会对所有剩余数值列进行聚合。 |
   | **`index`**      | 用作新 DataFrame 的行索引（分组键）。可以是单个列名，也可以是多个列名的列表，从而创建层次化索引。 |
   | **`columns`**    | 用作新 DataFrame 的列索引（分组键）。可以是单个列名或多个列名，形成列上的层次化索引。 |
   | **`aggfunc`**    | 聚合函数。可以是字符串（如 `'sum'`, `'mean'`, `'count'`），函数（如 `np.sum`），或为每个 `values`
   列指定不同函数的字典（例如 `{'销售额': 'sum', '利润': 'mean'}`）。 |
   | **`fill_value`** | 标量，用于替换结果中的缺失值（NaN）。 |
   | **`margins`**    | 布尔值，默认为 `False`。若为 `True`，会在行和列末尾添加总计（小计），可通过 `margins_name`
   自定义总计标签。 |
   | **`dropna`**     | 布尔值，默认为 `True`。是否删除全部为 NaN 的列。 |
   | **`observed`**   | 布尔值，默认为 `False`。当使用分类数据时，如果设为 `True`
   ，只显示实际出现在数据中的组合；否则会显示所有可能的组合。 |
