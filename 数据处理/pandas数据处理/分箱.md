## 1. 基本语法

python

```
import pandas as pd

pd.cut(
    x,                # 一维数组（Series 或 list），要分箱的连续数据
    bins,             # 分箱规则：整数（等宽箱数）或序列（自定义区间边界）
    right=True,       # 区间是否右闭（默认 True，即区间右端点包含在内）
    labels=None,      # 分箱后的标签（需与箱数一致），若不指定则返回区间对象
    retbins=False,    # 是否返回分箱边界，默认为 False
    precision=3,      # 区间标签的小数精度，默认为 3
    include_lowest=False, # 第一个区间是否包含左端点，默认为 False
    duplicates='raise',   # 如何处理重复的 bins 边界：'raise' 报错，'drop' 丢弃
    ordered=True      # 标签是否有序，默认为 True（分类数据的有序性）
)
```

------

## 2. 主要参数详解

| 参数                   | 说明                                                                                                                                                 |
|:---------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------|
| **`x`**              | 要分箱的输入数据，可以是 Series、数组或列表。                                                                                                                         |
| **`bins`**           | **核心参数**：定义分箱方式。 - 若为**整数**（如 `bins=4`），则将数据均匀划分为 `bins` 个等宽区间（范围由最小值到最大值）。 - 若为**序列**（如 `[0, 18, 35, 60, 100]`），则指定自定义区间的边界，左闭右开（取决于 `right` 参数）。 |
| **`right`**          | 布尔值，默认为 `True`，表示区间右侧闭合（包含右端点）；若为 `False`，则左侧闭合。                                                                                                   |
| **`labels`**         | 指定每个区间的标签，可以是数组或列表，长度必须与箱数相等。若为 `False`，则只返回整数编码（0,1,2...）。                                                                                        |
| **`retbins`**        | 布尔值，若为 `True`，则除了返回分箱后的 Series 外，还返回分箱边界数组，便于后续使用。                                                                                                 |
| **`include_lowest`** | 布尔值，默认为 `False`，是否包含第一个区间的左端点。若为 `True`，则第一个区间为左闭。                                                                                                 |
| **`duplicates`**     | 当 `bins` 序列中出现重复边界时的处理方式：`'raise'`（报错）或 `'drop'`（丢弃重复，保留唯一边界）。                                                                                     |
| **`ordered`**        | 生成的分类数据是否有序，默认为 `True`（适合等级变量）。                                                                                                                    |

------

## 3. 返回值

- 默认返回一个 **Categorical Series**（分类类型 Series），每个元素所属的区间（Interval 对象或标签）。
- 若指定了 `labels`，则返回对应标签的分类数据。
- 若 `retbins=True`，则返回 `(out, bins)` 元组，其中 `out` 为分类 Series，`bins` 为计算或传入的区间边界。

## 4. 详解

1. 示例数据

```python
ages = [20, 22, 25, 27, 30, 35, 40, 45, 50, 55, 60]

```

2. 等宽区间

```python
import pandas as pd
import numpy as np

age_groups = pd.cut(ages, bins=3)
```

直接设置bins，对已有数据进行等分。

3. 自定义区间边界（如年龄段）


```python
import pandas as pd
bins = [0, 18, 35, 50, 100]  # 自定义分界点
labels = ['少年', '青年', '中年', '老年']  # 对应标签
age_groups = pd.cut(ages, bins=bins, labels=labels)
```
区间映射为 0-18，18-35，35-50，50-100并对应labels中数据。注意默认区间是**左开右闭**

4. 返回整数编码（不指定 labels）
```python
codes = pd.cut(ages, bins=bins, labels=False)  # labels=False 返回整数编码
print(codes) 

```
