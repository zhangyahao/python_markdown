1. 基本语法

```python
DataFrame.shift(periods=1, freq=None, axis=0, fill_value=None)
Series.shift(periods=1, freq=None, axis=0, fill_value=None)
```

2. 作用
   获取前一条或后一条数据，沿指定轴方向
3. 参数详解

| 参数               | 说明                                                                   |
|:-----------------|:---------------------------------------------------------------------|
| **`periods`**    | 整数，表示移动的步数。正数表示向下/向后移动（索引不变，数据下移），负数表示向上/向前移动（数据上移）。默认为 1。           |
| **`freq`**       | 仅对时间序列有效。如果指定了 `freq`，则移动的是**时间索引**而非数据本身，相当于将整个时间索引向前或向后移动指定的时间偏移量。 |
| **`axis`**       | 移动的轴：`0` 表示沿行方向移动（默认），`1` 表示沿列方向移动（即水平移动）。                           |
| **`fill_value`** | 用于填充移动后空缺位置的标量值，默认为 NaN。                                             |
