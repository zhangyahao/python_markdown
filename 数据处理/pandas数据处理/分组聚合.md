使用groupby进行数据分组，获取到的数据进行聚合。

1. 基本语法

```python
df.groupby(by=None, axis=0, level=None, as_index=True,
           sort=True, group_keys=True,
           observed=False, dropna=True)
```

2. 常用参数：

- by：用于分组的键，可以是列名、多个列名的列表、Series、字典、函数等。
- axis：分组轴，0 表示行（默认），1 表示列。
- as_index：布尔值，默认为 True，表示是否将分组键作为结果的索引；False 则分组键会作为普通列。
- sort：是否对分组键排序，默认为 True。
- dropna：是否排除 NaN 值，默认为 True。

3. 聚合
    - 分组后，对获取的每行值进行聚合。常用的聚合函数：`sum()`、`mean()`、`median()`、`min()`、`max()`、`count()`、`size()`、
      `std()`、
      `var()`、`describe()`等
    - `agg()` 灵活聚合
        ```python
             # 对销售额求和，对利润求均值
         df.groupby('门店').agg({'销售额': 'sum', '利润': 'mean'})
         # 对销售额应用多个函数
         df.groupby('门店')['销售额'].agg(['sum', 'mean', 'max'])
         # 自定义函数
         df.groupby('门店')['销售额'].agg(lambda x: x.max() - x.min())  # 极差
        ```

4. 参数 as_index 的作用
   默认 as_index=True，分组键会成为结果的索引；若设为 False，则分组键作为普通列保留。当为True时，获取到的**数据格式**
   为Series本质为`SeriesGroupBy` ，为False时**数据格式**为dataframe。但是其本质为 `DataFrameGroupBy`。
   在一些版本中。如果对其结果进行`非聚合函数`操作时，比如 `unique`当希望获得一个`DataFrame`时，设置`as_index=False`
   会报错。可以不设置该参数，`unique`函数后，增加`.reset_index()`进行数据转换。
