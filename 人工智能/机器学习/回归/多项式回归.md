1. 什么是多项式回归  
   多项式回归是线性回归的一种扩展形式。当数据之间存在非线性关系时，简单的线性模型无法很好地拟合。
   多项式回归通过引入自变量的高次项（如平方、立方等），使模型能够捕捉数据的曲线趋势。  
   数学上，多项式回归模型仍然是一个线性模型,但特征空间被扩展为包含自变量的幂次项。例如一元多项式回归：
   *y*=*β*0+*β*1*x*+*β*2*x*2+⋯+*β**d**x**d*+*ε*  
   这里 $d$ 是多项式的阶数（degree）。对于多元特征，可以包含各特征的幂次以及它们之间的交互项（如 $x_1 x_2$、$x_1^2 x_2$ 等）。
2. 为什么需要多项式回归
    - 真实世界中的数据往往不是简单的直线关系，比如
        - 经济增长与时间（前期增长快，后期趋缓）
        - 药物剂量与疗效（先升后降）
        - 产品价格与销量（可能呈非线性）
    - 多项式回归提供了一个简单而强大的方式来拟合这些非线性模式，而无需使用更复杂的非参数模型
3. python实现

    ```python
    import numpy as np
    import matplotlib.pyplot as plt
    from sklearn.preprocessing import PolynomialFeatures
    from sklearn.linear_model import LinearRegression
    from sklearn.pipeline import make_pipeline
    
    # 生成非线性数据
    np.random.seed(42)
    X = np.linspace(-3, 3, 100).reshape(-1, 1)
    y = 0.5 * X ** 2 + X + 2 + np.random.randn(100, 1) * 0.5
    
    # 创建多项式回归管道（degree=2）
    poly_model = make_pipeline(PolynomialFeatures(degree=2), LinearRegression())
    poly_model.fit(X, y)
    
    # 预测并绘图
    X_test = np.linspace(-3.5, 3.5, 200).reshape(-1, 1)
    y_pred = poly_model.predict(X_test)
    
    plt.scatter(X, y, alpha=0.5, label='真实数据')
    plt.plot(X_test, y_pred, 'r', label='多项式拟合 (degree=2)')
    plt.xlabel('X')
    plt.ylabel('y')
    plt.legend()
    plt.show()
    
    ```

4. 查看多项式系数

    ```python
    # 获取管道中的步骤
    poly = poly_model.named_steps['polynomialfeatures']
    lr = poly_model.named_steps['linearregression']
    
    # 获取特征名称（假设原始特征名为 'x'）
    feature_names = poly.get_feature_names_out(input_features=['x'])
    coef = lr.coef_.flatten()
    intercept = lr.intercept_
    
    print("特征名称:", feature_names)
    print("系数:", coef)
    print("截距:", intercept)
    ```

5. 多项式回归的关键参数

   * **degree**：多项式的最高次数。过小的 degree 会导致欠拟合，过大的 degree 容易过拟合。
   * **include_bias**：是否添加常数项（值为 1 的列）。如果设置为 True，则线性回归中应设置 fit_intercept=False 以避免重复截距；推荐使用
     include_bias=False，由线性回归自己处理截距。
   * **interaction_only**：如果为 True，则只生成交互项（如 $x_1 x_2$），不生成各特征的平方、立方等（默认为 False）。

6. 多项式回归的局限

   * **外推能力差**：多项式在数据范围之外会迅速发散，因此不适合做超出训练范围的预测。
   * **特征爆炸**：当原始特征很多且 degree 较高时，特征数量会呈指数增长（组合数）。例如，10个特征，degree=3，特征数量可达 286
     个，容易导致维度灾难。
   * **数值稳定性**：高次项可能导致数值不稳定，通常需要对特征进行标准化

7. 总结

   - 多项式回归是线性回归处理非线性关系的简单有效方法。
   - 通过 `PolynomialFeatures` + `LinearRegression` 实现，但需警惕过拟合。
   - 使用交叉验证选择阶数，并结合正则化（岭回归 / Lasso）提高泛化能力。
   - 管道（Pipeline）可以优雅地封装整个流程，方便部署和调参。
