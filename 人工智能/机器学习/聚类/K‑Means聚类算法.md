## 核心思想

将数据划分为 K 个簇，使得簇内样本尽量紧凑（距离小），簇间样本尽量分离。通过迭代优化，使每个样本到其所属簇中心的距离平方和最小。

## 算法步骤

1. 初始化：随机选择 K 个样本作为初始簇中心（质心）。
2. 分配：计算每个样本到 K 个中心的距离，将其归入距离最近的中心所在的簇。
3. 更新：重新计算每个簇内所有样本的均值，作为新的簇中心。
4. 迭代：重复步骤 2 和 3，直到簇中心不再变化或达到最大迭代次数。

## 关键参数

| 参数             | 类型               | 默认值           | 描述                                                          |
|:---------------|:-----------------|:--------------|:------------------------------------------------------------|
| `n_clusters`   | int              | 8             | **聚类的个数（K 值）**。必须指定。                                        |
| `init`         | str 或 array-like | `'k-means++'` | 初始化方法：`'k-means++'`（智能分散初始中心）、`'random'`（随机选择）、或用户提供初始中心矩阵。 |
| `n_init`       | int 或 `'auto'`   | `'auto'`      | 随机初始化运行的次数，最终选取 inertia 最小的结果。`'auto'` 会根据数据量自动选择（通常 10 次）。 |
| `max_iter`     | int              | 300           | 单次初始化中最大迭代次数。                                               |
| `tol`          | float            | 1e-4          | 收敛阈值，当簇中心变化小于该值时停止迭代。                                       |
| `algorithm`    | str              | `'lloyd'`     | 迭代算法：`'lloyd'`（经典 EM 风格）、`'elkan'`（使用三角不等式加速，适合稠密数据）。       |
| `random_state` | int 或 `None`     | `None`        | 随机种子，用于控制初始化随机性，保证结果可复现。                                    |
| `verbose`      | int              | 0             | 控制日志输出（0 静默，1 输出迭代信息）。                                      |
| `copy_x`       | bool             | `True`        | 是否复制输入数据（若 `False`，原始数据可能被修改）。                              |

## 优缺点

| 优点                             | 缺点                    |
|:-------------------------------|:----------------------|
| 简单、快速，适合大规模数据                  | 必须事先指定 K 值            |
| 对球形簇效果好                        | 对异常值敏感                |
| 可解释性强                          | 容易陷入局部最优              |
| 可扩展至大数据集（如 Mini‑Batch K‑Means） | 只适用于数值型数据，不适合发现非凸形状的簇 |

## 实现
```python
from sklearn.cluster import KMeans
import numpy as np

X = np.random.rand(100, 2)  # 100个二维样本
kmeans = KMeans(n_clusters=3, random_state=42, n_init='auto')
kmeans.fit(X)

labels = kmeans.labels_          # 每个样本的簇标签
centers = kmeans.cluster_centers_ # 簇中心
inertia = kmeans.inertia_        # 簇内平方和（总距离）

# 预测新样本
new_sample = [[0.5, 0.5]]
pred = kmeans.predict(new_sample)
```
## 注意事项

- 特征缩放：K‑Means 依赖距离，不同量纲的特征会严重影响结果，务必进行标准化（如 StandardScaler）。
- 异常值处理：离群点会严重干扰簇中心，建议先检测并处理异常值。
- 多次运行：使用 k-means++ 或多尝试几次随机初始化，避免局部最优。
