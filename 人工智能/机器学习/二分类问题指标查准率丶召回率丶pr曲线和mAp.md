### 1. 查准率（Precision）与召回率（Recall）

假设有二分类问题，混淆矩阵如下：

|      | 预测正例 | 预测负例 |
|:-----|:-----|:-----|
| 真实正例 | TP   | FN   |
| 真实负例 | FP   | TN   |

- **查准率**（Precision） = $$ \frac{TP}{TP+FP} $$

  预测为正的样本中，实际为正的比例。衡量“模型有多准”。
- **召回率**（Recall） = $$ \frac{TP} {TP+FN} $$
  实际为正的样本中，被正确预测的比例。衡量“模型找得多全”。

两者往往相互制约：提高阈值会提升查准率但降低召回率，反之亦然。

### 2. PR 曲线

**PR 曲线**：以召回率 R 为横轴，查准率 P 为纵轴，通过改变分类阈值（或排序位置）绘制的曲线。

**如何获取 PR 曲线的点**（以二分类为例）：

1. 模型输出每个样本的**正类概率**（或置信度）。
2. 将所有样本按概率从高到低排序。
3. 依次将每个样本的预测概率作为**阈值**：
    - 大于等于阈值的视为正例，其余为负例。
    - 计算当前阈值下的 (Precision, Recall) 对。
4. 将这些点连成曲线（通常用阶梯状或插值平滑）。

### 3. AP（Average Precision，平均精度）

AP 是 PR 曲线下的面积（Area Under PR Curve），用来**量化模型在单个类别上的综合性能**。

### 4. mAP（mean Average Precision，平均精度均值）

**mAP** 是多类别分类/检测任务中的标准指标：

$$  mAP= \frac{1}{C} \sum_{i=1}^{C} AP_i$$
其中C为类别总数， $AP_i$为第i个值。

- **图像分类**：对每个类别独立计算 PR 曲线和 AP，再取平均。
- **目标检测**：还需考虑 IoU（交并比）阈值（如 VOC 用 0.5，COCO 用 0.5:0.95 多个阈值取平均）。
