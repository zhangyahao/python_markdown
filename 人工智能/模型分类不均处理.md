## 一、数据层面（重采样）

### 1. 欠采样（Undersampling）

减少多数类样本，使与少数类数量相当。

- **随机欠采样**：随机删除多数类样本。简单但可能丢失重要信息。
- **NearMiss**：根据多数类样本与少数类样本的距离选择保留样本。
- **Tomek Links**：删除边界上的多数类样本，使类别间分界更清晰。

### 2. 过采样（Oversampling）

增加少数类样本。

- **随机过采样**：复制少数类样本。易导致过拟合。
- **SMOTE**（Synthetic Minority Over‑sampling Technique）：在少数类样本之间插值生成新样本，缓解过拟合。
- **Borderline‑SMOTE**：只在边界附近的少数类样本间插值。
- **ADASYN**：根据分布自适应地生成更多边界样本。

### 3. 混合采样

先过采样再欠采样（如 SMOTE + Tomek Links）。

### 4. 数据增强（针对图像/文本）

对少数类样本做旋转、裁剪、加噪、同义词替换等。

## 二、算法层面

### 1. 代价敏感学习（Cost‑Sensitive Learning）

为不同类别的误分类赋予不同代价。常用方法：

- **类别权重**：`class_weight='balanced'`（如 sklearn、XGBoost 中）自动按样本数倒数设置权重。
- **手动代价矩阵**：`cost_matrix[0][1]` 大于 `cost_matrix[1][0]` 使模型更重视少数类。

### 2. 集成方法

- **Bagging 变体**：BalancedBaggingClassifier（每轮 bootstrap 采样保持类别均衡）。
- **Boosting 变体**：
    - **SMOTEBoost**：每轮 AdaBoost 前用 SMOTE 生成少数类样本。
    - **RUSBoost**：每轮前进行随机欠采样。
    - **EasyEnsemble / BalanceCascade**：多次欠采样 + AdaBoost 集成。

### 3. 单类分类（极端不平衡）

若少数类极其稀少（如异常检测），可训练**单类 SVM**、**隔离森林**（Isolation Forest）或**自编码器**，仅学习多数类的分布。

### 4. 异常检测专用方法

- One‑Class SVM
- 自编码器（重构误差大者为异常）
- 孤立森林

## 三、评估与阈值调整

### 1. 评估指标（避免使用准确率）

- **查准率、召回率、F1‑score**
- **PR 曲线下的面积（AP）**
- **AUC‑ROC**（对不平衡也较稳健）
- **G‑mean** =$\sqrt{召回率_+ \times 召回率_- } $ (正负召回率)

### 2. 调整分类阈值

模型默认阈值 0.5 往往不适合不平衡数据。通过验证集寻找最优阈值，使 F1 或 G‑mean 最大。
