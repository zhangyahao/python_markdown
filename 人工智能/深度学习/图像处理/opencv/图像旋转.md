1. **仿射变化：最灵活的旋转方法**
   实现任意角度、任意中心点旋转最推荐的方法。
    1. 获取旋转矩阵 `cv2.getRotationMatrix2D(center, angle, scale)`
       - `center`: 旋转中心点坐标，用 (x, y) 元组表示
       - `angle`: 旋转角度（单位：度）。正值表示逆时针旋转，负值表示顺时针旋转
       - `scale`: 图像缩放比例，1.0 表示不缩放。
    2. 应用旋转： `cv2.warpAffine(src, M, dsize, flags, borderMode)`
       - `src`: 输入图像
       - `M`: 第一步生成的旋转矩阵
       - `dsize`: 输出图像的尺寸，格式为 (宽度, 高度)。
       - `flags`: 插值方法，默认为 cv2.INTER_LINEAR（双线性插值）。
       - `borderMode`: 边界填充模式，默认为 cv2.BORDER_CONSTANT，可通过 borderValue 设置填充颜色，避免黑边。
       
2. **快速旋转：cv2.rotate()**      
   对于90°、180°等特定角度旋转，cv2.rotate() 是最简便的方法。它使用预定义的 RotateFlags 枚举值，直接完成旋转。
   ```python
       import cv2

       img = cv2.imread('your_image.jpg')
       
       # 顺时针旋转90度
       rotated_90_cw = cv2.rotate(img, cv2.ROTATE_90_CLOCKWISE)
       
       # 逆时针旋转90度
       rotated_90_ccw = cv2.rotate(img, cv2.ROTATE_90_COUNTERCLOCKWISE)
       
       # 旋转180度
       rotated_180 = cv2.rotate(img, cv2.ROTATE_180)
   ```
