异常值包括空值（缺失值），重复值，替补值

### 空值处理

- 查找
  `Series/DataFrame.isnull/isna()`
- 处理

* 删除

```python
  df.dropna(
    axis=0,  # 删除的轴的方向
    how='any',  # 删除的条件
    subset=None,  # 指定某个行或者列
    inplace=False,  # 是否在原DataFrame上替换
)
```

`how`:接收特定string。表示删除的形式。any表示只要有缺失值存在就执行删除操作。all表示当且仅当全部为缺失值时执行删除操作。默认为any
注意：**删除轴**和**subset** 是**垂直**的，比如使用axis=0，删除行，那么就应该将列名作为参数传递确定数据位置。

* 替换

```python
df.fillna(
    value=None,  # 替换值
    method=None,  # 填充方式
    axis=None,  # 填充方向
    inplace=False,  # 是否在原DataFrame上替换
    limit=None,  # 填充缺失值的个数
)
```

`method`:接收特定string。backfill或bfill表示使用下一个非缺失值填补缺失值。pad或ffill表示使用上一个非缺失值填补缺失值。默认为None。

### 重复值处理

- 查找

```python
DataFrame.duplicated(
    subset=None,  # 对所在的列进行检测
    keep='first'  # 保留的数据的选项
)
```

`keep`:有三个可选参数，分别是 first、last、False，默认为first，表示只保留第一次出现的重复项,删除其余重复项，last
表示只保留最后一次出现的重复项，False 则表示删除所有重复项

- 处理
```python
student.drop_duplicates(
        subset=None, # 删除所在的列
        keep='first', # 保留的数据的选项
        inplace=False, # 是否在原有的DataFrame上操作
 )
```
