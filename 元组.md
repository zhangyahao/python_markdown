| 特性类别     | 具体内容      | 说明与示例                                               |
|:---------|:----------|:----------------------------------------------------|
| **基本定义** | 不可变有序序列   | 使用圆括号 `()` 创建，元素用逗号分隔 `tuple1 = (1, 2, 3, "hello")` |
| **创建方式** | 直接创建      | `t = (1, 2, 3)` `t = 1, 2, 3` (省略括号)                |
|          | 单元素元组     | `t = (1,)` (必须有逗号，否则不是元组)                           |
|          | 空元组       | `t = ()`                                            |
|          | 使用tuple() | `t = tuple([1, 2, 3])` (将其他序列转为元组)                  |
| **核心特性** | 不可变性      | 创建后不能修改、添加、删除元素 `t[0] = 5` ❌ 会报错                    |
|          | 有序性       | 元素按定义顺序存储，可通过索引访问                                   |
|          | 可哈希性      | 可作为字典的键 `dict_tuple = {(1,2): "value"}`             |
|          | 可包含任意类型   | 可包含数字、字符串、列表、甚至其他元组                                 |
| **基本操作** | 索引访问      | `t[0]` (访问第一个元素) `t[-1]` (访问最后一个元素)                 |
|          | 切片操作      | `t[1:3]` (获取第2到第3个元素)                               |
|          | 长度获取      | `len(t)`                                            |
|          | 成员检查      | `element in t`                                      |
| **优势**   | 性能优化      | 比列表占用内存少，访问速度快                                      |
|          | 数据安全      | 防止意外修改，保证数据完整性                                      |
|          | 字典键值      | 可作为字典的键使用                                           |
| **常用方法** | count()   | 统计元素出现次数 `t.count(2)`                               |
|          | index()   | 查找元素首次出现的位置 `t.index("hello")`                      |
| **应用场景** | 函数返回多个值   | `def get_user(): return name, age, email`           |
|          | 字典键       | 坐标点：`point = (x, y)`                                |
|          | 数据记录      | 保证数据不被修改的场景                                         |
