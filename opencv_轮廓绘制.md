### 轮廓绘制
1. 读取图片源
2. 灰度处理
3. 二值化
4. 查找轮廓   
    ```python
          contours,hierarchy =cv2.findContours(
                     # 二值化图片源
                     img_binary,
                     # 轮廓查找模式/形式。确定轮廓范围
                     cv2.RETR_EXTERNAL,
                     cv2.CHAIN_APPROX_NONE # 轮廓点保存方法，cv2.CHAIN_APPROX_NONE所有轮廓点 CHAIN_APPROX_SIMPLE轮廓点端点
                      )
   ```
    * 轮廓查找模式/形式
       1. cv2.RETR_EXTERNAL - 只检测最外层轮廓
           * 特点：只返回最外层的轮廓，忽略所有内部嵌套的轮廓
           * 适用场景：当你只关心物体外部边界时
           * 层次关系：所有轮廓都是平等的，没有父子关系
       2. cv2.RETR_LIST - 检测所有轮廓，不建立层次关系
           * 特点：检测所有轮廓，但不创建父子层次关系
           * 适用场景：只需要所有轮廓，不关心嵌套关系
           * 层次关系：所有轮廓都是同一级别
       3. cv2.RETR_CCOMP - 检测所有轮廓，建立两级层次
           * 特点：将所有轮廓组织为两级层次结构
           * 层次关系：
               * 第一级：所有外部轮廓（没有父轮廓）
               * 第二级：所有内部轮廓（孔洞等）
       4. cv2.RETR_TREE - 检测所有轮廓，建立完整层次树
           * 特点：建立完整的轮廓层次树，包含所有嵌套关系
           * 适用场景：需要精确了解轮廓嵌套关系时
           * 层次关系：完整的父子关系树
       5. 层次关系理解 hierarchy
          `hierarchy` 是一个数组，每个轮廓对应一个四元素数组：[next, previous, first_child, parent]
           - next: 同一层级的下一个轮廓索引
           - previous: 同一层级的前一个轮廓索引
           - first_child: 第一个子轮廓索引
           - parent: 父轮廓索引
             如果值为 -1，表示不存在该关系。
       6. 实用建议
           1. 物体计数 → 使用 RETR_EXTERNAL
           2. 简单轮廓提取 → 使用 RETR_LIST
           3. 处理带孔洞的物体 → 使用 RETR_CCOMP
           4. 分析复杂嵌套结构 → 使用 RETR_TREE
   5. 绘制轮廓
        ```python
            im_cnt=cv2.drawContours(im_gary, # 图片源，使用该函数后，会对该参数进行修改,一般使用原图，或者复制原图进行绘制，
                                             # 因为灰度图为单通道，轮廓颜色一般设置为三通道无法有效展示,
                                             # 若使用灰度图，轮廓线一般为不同深浅颜色的灰色
                contours, # 轮廓点
                -1,# 轮廓缩影，若图片源中存在多个单个物体，-1代表所有。若要单独展示一个或多个，指定索引
                (255,0,255), # 轮廓颜色  bgr元组
                2 # 线宽 大于0是线宽，若为-1则填充内部
                     )
        ```
