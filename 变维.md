# 视图变维 np.reshape    
  * 保持数组数量不变，重新排列数据维度
  * 参数设为-1时，维度自动推算，例如
    ```python
        import numpy as np
        arr1=np.arange(1,9)
        arr2=arr1.reshape(2,2,2)
        arr3=arr2.reshape(2,4)
        arr4=arr2.reshape(-1,4)
        arr5=arr1.reshape(-1,2,2)
    ```  
  * 函数运行结果为视图，源数据变化视图一起变化
# 复制变维 np.flatten

| 参数    | 可选值 | 说明                                   | 适用场景              |
|-------|-----|--------------------------------------|-------------------|
| order | 'C' | (默认) 	按行优先（Row-major order）顺序展平	     | 常规处理，符合大多数语言习惯    |
|       | 'F' | 按列优先（Column-major order）顺序展平	        | 处理Fortran风格数据或列操作 |
|       | 'A' | 保留原数组的内存顺序展平（尽可能按列优先如果内存中为Fortran连续） | 保持原始存储效率时         |
|       | 'K' | 按元素在内存中出现顺序展平	                       | 需要保持特定内存布局时       |
