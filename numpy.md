# list ，array和ndarray区别  

| 特性       | 列表 (List)              | 数组 (Array)                  | N维数组 (Ndarray)      |
|:---------|:-----------------------|:----------------------------|:--------------------|
| **所属模块** | Python 内置              | `array` 模块                  | NumPy 库             |
| **数据类型** | 异构（任意类型）               | 同构（相同数值类型）                  | 同构（相同数值类型）          |
| **维度**   | 一维（可嵌套模拟多维）            | 严格一维                        | 支持任意维度              |
| **内存效率** | 较低（存储对象引用）             | 较高（连续内存）                    | 最高（优化连续内存）          |
| **性能**   | 通用操作灵活                 | 数值操作较快                      | 数值计算极快（向量化）         |
| **功能方法** | 丰富（append, pop, sort等） | 有限（基础数值操作）                  | 强大（数学运算、线性代数等）      |
| **创建方式** | `[1, "a", [2,3]]`      | `array.array('i', [1,2,3])` | `np.array([1,2,3])` |
| **适用场景** | 通用数据存储                 | 简单数值存储                      | 科学计算、数据分析           |
